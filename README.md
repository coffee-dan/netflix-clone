# Netflix Clone

Quick start:

```
$ yarn # npm install
$ yarn build # npm run build
```

## Development

Run Webpack in watch-mode to continually compile the JavaScript and output to [localhost:8080](http://localhost:8080/) as you work:

```
$ yarn start # npm run start
```
